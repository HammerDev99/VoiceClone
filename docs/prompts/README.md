# docs/prompts/

Prompts de orquestación para sesiones agénticas (CDAID v2). Guarda aquí los
prompts reutilizables que lanzan auditorías, sprints o tareas con agentes.

Ejemplo de uso: un prompt para "auditoría multi-instrumento de la fase X"
que invoque `/refactoring` y `/design-patterns` sobre módulos concretos.
