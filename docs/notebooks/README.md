# docs/notebooks/

Mini-proyectos exploratorios (Jupyter) para experimentar antes de migrar a
`src/`. Ej.: probar settings de voz (estabilidad/similaridad), comparar
formatos de salida, o calibrar la longitud de respuesta de la persona.

Flujo CDAID: explorar aquí → validar → migrar el código estable a `src/voiceclone/`.
